## Key Bindings:

- Bold: "Ctrl + B",
- Italic: "Ctrl + I",
- Underline: "Ctrl + U",
- Heading: "Ctrl + H",
- Paragraph: "Ctrl + P",
- List: "Ctrl + l",
- Line: "Ctrl + I",
- Enter: "Enter",

### Stack used: React Vite, TypeScript, SASS, Axios, Tanstack Query
